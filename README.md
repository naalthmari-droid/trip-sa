# TRIPSA - Smart Tourism Recommendation System for Saudi Arabia

A Streamlit-based application that generates personalized trip plans for tourists visiting Saudi Arabia.

## Features
- Interactive questionnaire for travel preferences
- Smart activity scheduling based on activity type and duration
- Multi-city itinerary with transportation recommendations
- Hotel recommendations from Booking.com data
- PDF export of trip plans with Arabic text support

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Data Sources
- **Visit Saudi** - Tourist attractions dataset
- **Booking.com** - Hotel accommodations dataset
- **Ministry of Tourism** - Survey data
